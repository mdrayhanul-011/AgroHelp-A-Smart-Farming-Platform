AgroHelp is a smart, user-friendly digital agriculture platform designed to support farmers in Bangladesh. It provides crop suggestions, step-by-step farming guidance, cost estimation, and real-time market insights. The platform also includes AI-powered agricultural assistance through Gemini API chat to help farmers make data-driven and sustainable farming decisions.

## Tech Stack
- React + Tailwind CSS
- Node.js + Express
- MongoDB (Atlas)
- JWT Authentication
- Gemini AI API

## Purpose
Empowering farmers with accessible technology to improve productivity, reduce costs, and promote sustainable agriculture.
